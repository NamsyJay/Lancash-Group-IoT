## Printer Dashboard Visualizer

### GOAL
Develop a web application that provides real-time insights into the performance of IoT-enabled devices at Lancash.

### PROJECT SCOPE
- Real-Time Monitoring: Displays device metrics like operational status and anomaly detection.
- Interactive Visualisations: Use chart.js to present data trends.
- Scalable Architecture: Utilise AWS services like Lambda, API Gateway and DynamoDB for backend support.
- Secure Access: ensures that data is appropriately retrieved across AWS services.
